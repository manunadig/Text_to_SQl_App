## Text to SQL App Master

## Overview

Text to SQL App Master is an intelligent application that converts natural language text queries into structured SQL queries. It allows users to interact with relational databases using simple English sentences instead of writing complex SQL code.

This project bridges the gap between non-technical users and database systems by leveraging Natural Language Processing (NLP) techniques to generate accurate SQL queries automatically.

## Features

Converts English text queries into SQL queries
Supports common SQL operations:
*SELECT
*WHERE
*ORDER BY
*GROUP BY
*Basic JOIN

Reduces manual SQL coding effort
Easy to integrate with web applications
Beginner-friendly and extensible

## Installation & Setup

## 1.Clone the Repository
git clone https://github.com/your-username/Text-to-SQL-App-Master.git
cd Text-to-SQL-App-Master

## 2.Create Virtual Environment (Optional but Recommended)
python -m venv venv


Activate environment

Windows:
venv\Scripts\activate

Linux / Mac:
source venv/bin/activate

## 3.Install Dependencies
pip install -r requirements.txt

## How to Run the Application
Step 1: Start the Flask Server
python app.py

Step 2: Open Browser

Go to:

http://127.0.0.1:5000
